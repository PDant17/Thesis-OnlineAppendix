# The evolution of Game Software Engineering: A Systematic Literature Review
## Online Appendix
**Author:** Pietro D'Antuono, University of Salerno, Italy

**Degree Program:** Bachelor's Degree in Computer Science

**Academic Year:** 2024-25

**Supervisor:** Prof. Fabio Palomba

This is the online appendix for the bachelor's thesis titled _The evolution of Game Software Engineering: A Systematic Literature Review_. It provides all supplementary materials used during the review process to ensure transparency and reproducibility.

The repository contains the following files:
- **PRISMA Diagram.pdf** - flow diagram for the selection process;
- **Irrelevant Studies.xlsx** - studies excluded during the title and abstract screening phase;
- **Excluded Studies.xlsx** - studies excluded during the full-text screening phase;
- **Included Studies.xlsx** - the final set of studies included in the review;
- **Data Extraction.xlsx** - all extracted data from the included studies.
